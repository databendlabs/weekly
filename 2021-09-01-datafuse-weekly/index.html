This week in Datafuse #5

2021-09-01

Datafuse is an open source elastic and scalable cloud warehouse, it offers blazing fast query and combines elasticity, simplicity, low cost of the cloud, built to make the Data Cloud easy.

Big changes

Below is a list of some major changes that we don’t want you to miss.

  • custom logic types on arrow2 by @sundy-li. (#1648)
  • add a basic lru memory/disk cache by @PsiACE (#1600).
  • use custom HashMap for HashMethodKind::Serializer by @zhang2014. (#1500)
  • a series of refactors on catalog by @BohuTANG. (#1642, #1643, #1644)
  • cross compile for arm-based architecture by @ZhiHanZ. (#1689)
  • refactor some warp services to axum by @ZhiHanZ. (#1654)
