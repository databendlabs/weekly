Installion

This document describes how to build and run FuseQuery as a distributed query engine.

1. Deploy#

  • Run with Docker(Recommended)
  • Release binary
  • From source
  $ docker pull datafuselabs/datafuse  $ docker run --init --rm -p3307:3307 datafuselabs/datafuse
$ curl -fsS https://raw.githubusercontent.com/datafuselabs/datafuse/master/scripts/installer/install-datafuse.sh | bash
$ git clone https://github.com/datafuselabs/datafuse.git$ cd datafuse$ make setup$ make run
